Detailed Description

App meant to define and drive token generation for traffic control.

Defines token bucket capacity, and generation rate for strategy level token bucket objects; one for each of the priority queues. Also sends token generation commands to token bucket at the given generation rate.

Uses references provided by TBucketRef class in order to communicate with the token buckets that reside at the strategy level.
